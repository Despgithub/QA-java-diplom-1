# Diplom_1

<hr>
Дипломный проект по Unit - тестированию.

## Описание

<hr>
На проекте используется Java 11 и Maven и следующие библиотеки:
<ul>
<li>Junit 4.13.2</li>
<li>Mockito 5.5.0</li>
<li>Jacoco 0.8.10</li>
</ul>

## Запуск тестов

<hr>

```
mvn clean test
```

## Получение отчета о покрытии

<hr>

```
mvn verify
```
